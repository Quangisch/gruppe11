Manual

- movement with arrow keys
- F 		sprint
- D		punch
- S		cast Spell
- E		toogle Spell
- Space 	interact
- 1,2,3,4	use Potions/select Options

- Esc		ingameMenu
-ingameMenu- CTRL + 1/2	save game at slot 1/2
-ingameMenu- 1/2	load game from slot 1/2
- C 		toggle camera fixed/automatic
- V		center camera in fixed-camera mode


- 8		toogle music on/off
- 9		toogle atmo on/off
- 0		reset game to mainMenu

- K		spawn coins
- L		spawn lifeOrbs
- M		spawn manaOrbs

- B 		show borders and hitboxes
- O 		get coordinates
- P		print message
